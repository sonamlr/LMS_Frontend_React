# LMS Frontend

### Setting up instruction

1. Clone Project

```
    git clone https://github.com/sonamlr/LSM_Frontend_React.git

```

2. Move into Directory

```
    cd LSM_Frontend_React

```

3. Install Dependencies

```

    npm install

```

4. Run Server

```

    npm run dev

```
